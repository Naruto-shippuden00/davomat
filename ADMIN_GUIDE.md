## ➕ SINF QO'SHISH


- Katta-kichik harf farq qilmaydi (9-a ham ishlaydi)

## ➕ O'QITUVCHI QO'SHISH

### 1. O'qituvchi ID sini oladi:

O'qituvchiga aytasiz:
"Telegram'da @userinfobot ga /start yuboring va ID ni menga bering"

O'qituvchi oladigan ID misol: `987654321`

### 2. Admin botda qo'shadi:

```
/admin
👇
➕ O'qituvchi qo'shish
👇
987654321 (O'qituvchi ID si)
👇
Alisher Karimov (O'qituvchi ismi)
👇
✅ O'qituvchi qo'shildi!
```

### 3. O'qituvchi botga kiradi:

O'qituvchi endi botga `/start` yuborishi mumkin va davomat belgilaydi.

## ❓ TRIK VA MASLAHATLAR

### ✅ Birinchi qadamlar:

1. **Sinflarni qo'shing** (9-A, 9-B, 10-A...)
2. **Fanlarni qo'shing** (Matematika, Fizika...)
3. **O'quvchilarni qo'shing**
4. **O'qituvchilarni qo'shing**
5. **Tayyor!** O'qituvchilar davomat belgilashi mumkin
