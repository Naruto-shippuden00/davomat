## 📱 BOTGA KIRISH

### 1. Birinchi marta:

Admin sizni qo'shgandan keyin:

```
1. Botni oching
2. /start yuboring
3. Sizga "👨‍🏫 O'qituvchi" sifatida kirish beriladi
```

### Har bir o'quvchi uchun:

```

```


## ❓ TRIK VA MASLAHATLAR

### ✅ Eng yaxshi amaliyot:

1. **Dars boshida** davomat belgilang
2. **Kechikkanlar** uchun sabab so'rang
3. **Ota-onalarga xabar** yuborishni unutmang
4. **Haftalik hisobot** ni ko'rib turing

### ⏰ Vaqtni tejash:

- Bir necha o'quvchini ketma-ket belgilang
- Oxirida birgalikda saqlang
- Kechikish sabablari tayyor tugmalar bilan
